## Prerequisites

For development, you will need [Node.js](https://nodejs.org/) (v20+).

## Get started

Install the application dependencies.

```shell
npm install
```

Start the app.

```shell
npm start
```

To use collaboration features, start the peer server.

```shell
npm run peer-server
```
